# LinkShrink UI/UX Design Guide

## Design Goals

Create a modern, sleek SaaS experience that feels fast, professional,
and effortless on both desktop and mobile.

Prioritize: - Simplicity over complexity - Clear primary actions - Fast
navigation - High readability

------------------------------------------------------------------------

## Visual Style

### Theme

Dark mode only.

### Colors

-   Background: #0F1117
-   Surface: #1E2230
-   Text: #FFFFFF
-   Secondary Text: #B4BAC8
-   Primary Accent (Purple): #A855F7
-   Success Accent (Green): #22C55E

Use purple for actions and navigation. Use green for success states and
analytics highlights.

------------------------------------------------------------------------

## Typography

-   Font: Inter
-   Clean hierarchy
-   Generous spacing
-   Minimal text where possible

------------------------------------------------------------------------

## Layout

### Desktop

-   Centered content
-   Max width: 1200px
-   Dashboard-style card layouts

### Mobile

-   Single-column layout
-   Collapsible navigation
-   Touch-friendly controls

------------------------------------------------------------------------

## Components

### Cards

-   Rounded corners
-   Soft borders
-   Dark surfaces

### Buttons

-   Primary: Purple
-   Success: Green
-   Rounded and compact

### Inputs

-   Large, clear, and easy to scan
-   Purple focus states

------------------------------------------------------------------------

## User Experience

The URL input should be the primary focus of the application.

Users should be able to: 1. Paste a URL 2. Generate a short link 3. Copy
the result 4. Download a QR code

All within a few seconds.

------------------------------------------------------------------------

## Overall Feel

Modern SaaS. Clean dashboards. Minimal distractions.

Keywords:

**Fast • Clean • Professional • Minimal • Developer-Focused**
